System QC-cards for calibration. The calibration for each lot can be done by reading QC-cards for each batch of slides. Appliance stores in memory the last two batches of slides for each parameter. Identiﬁcation of parties and the slide type is automatic.

Function Automatic Dilution

Automating the tedious procedure of breeding. Breeding is carried out on board in automatic mode according to the established rate of dilution. Just set the mixing cups, vials with diluents and click "Breeding".

C-reactive protein. Calibration

Calibration for determination of c-reactive protein, consisting of three-dimension calibrating solutions, is carried out in automatic mode. You should only install the slides in the holder, and vials with calibrators in a special holder.
